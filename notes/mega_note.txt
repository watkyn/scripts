GDB notes (mac osx debugger)
GDB tips for the command line
po - print object ( calls description )
p - print (tell ref)
p *self (more details)
po <address of object>
c - continue
n - next line (step over)
s - step into
fin - return back up stack
bt - backtrace
up - up stack
down - down stack
